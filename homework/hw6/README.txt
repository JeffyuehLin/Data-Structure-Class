This is the sixth homework for Data-Structure Class. The program will sort data with the integers used with four sorting algorithms: selection sort, bubble sort, insertion sort, and quick sort.

The program will print out the results step by step. In the first three sorting algorithms, the program will also print how many steps in the algorithms.

The maximun size of input array is 10000 integers.
