This program is the fourth homework for Data-Structure Class. The program has two part. The first part is to implement a binary-search-tree using linked lists. The second part is to implement a treasure hunter game.

The first part of the program can insert, delete, search, and print by inorder and level order. 

The second part will read the map file to construct the maze, which is actually a binary search tree. User will input the key location, treasure location and a bomb number x is used to specify all the bombs with a number containing x. The program will delete the nodes which's number contains the bomb number, then finds a shortest path from the key to the treasure.
